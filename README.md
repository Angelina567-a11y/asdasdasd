## Описание программы
Это простое GUI-приложение для отслеживания личных расходов. Вы можете добавлять расходы, фильтровать их по категории и дате, а также видеть общую сумму за выбранный период.

## Как использовать
- Введите сумму, выберите категорию и дату, нажмите «Добавить расход».
- Для фильтрации выберите категорию или укажите диапазон дат, нажмите «Применить фильтр».
- Все данные сохраняются в файле `expenses.json`.

## Примеры использования
- Добавление расходов
- Фильтрация по категории: «Еда», «Транспорт» и др.
- Фильтрация по дате
